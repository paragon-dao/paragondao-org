# paragondao.org

Public site for ParagonDAO — the Health Economy protocol.

**Live:** https://paragondao.org

## Stack

React 19 · Vite 7 · Framer Motion · Magic.link auth · Cloudflare Pages

## Setup

```bash
pnpm install
cp env.example .env   # fill in your keys
pnpm dev
```

## Build

```bash
pnpm build
```
